A compilation of Kharkiv National University of Radio Electronics (NURE) Coursework tasks.

This particular repository consists of a set of simple programs designed to solve basic problems with the purpose of implementing algorithms learned during the course.
The core language used in this repository is C++, which is the first language taught at NURE in the Algorithmization and Programming course.

The current projects are created in Visual Studio.

List of Programming Languages (In-Progress): C++
